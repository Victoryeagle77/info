Mes réalisations


Voici ici quelques réalisations que j'ai effetué seul ou dans le cadre de ma scolarité en différents langages de programmation.

► SCRIPTING :

• VERIFICATEUR DE SAUVEGARDE :
Programme en BATCH-scripting pour l'entreprise de pharmacovigilance "BBAC" permettant de vérifier si une sauvegarde d'un répertoire d'un NAS à un autre a été correctement effectuée en renseignant sur le nombre de dossiers, de fichiers et le poids total en octets d'un répertoire.
== Télécharger le fichier NAS-vérification.bat ou observer le code du script ici ==

► LANGAGE C :

Les projets suivants utilisent une librairie graphique spécialement créée pour simplifier la librairie graphique lX11.
• TAQUIN :
Conception du jeu du Taquin, dans lequel vous choisissez une image qui se mélangera aléatoirement en puzzle de cases. A partir d'une case vide que vous pouvez déplacer d'une case en ligne, en colonne ou en diagonale, l'image doit être reconstituée. Celui-ci est éxecutable par la console commande avec l'environnement gcc et en makefile.
== Télécharger le contenu archivé du projet ici : Taquin.zip ==
• MASTERMIND :
Conception du jeu du Mastermind, dont le but est de reconstituer des combinaisons de couleurs générées aléatoirement, en plusieurs essais avec des indices de validité des couleurs placées. Celui-ci est éxecutable par la console commande avec l'environnement gcc et en makefile.
== Télécharger le contenu archivé du projet ici : Mastermind.zip ==
• MEMORY :
Conception du jeu du Memory, dont le but est de reconstituer des paires de cartes générées aléatoirement face cachées, il faut reconstituer toutes les paires de cartes générées dans la grille de jeu. Celui-ci est éxecutable par la console commande avec l'environnement gcc et en makefile.
== Télécharger le contenu archivé du projet ici : Memory.zip ==

► LANGAGE JAVA :

• DEMINEUR :
Conception du jeu du Démineur consistant à générer une grille avec des cases cachées sur lesquels l'utilisateurs doit cliquer. Certaines sont piégées par une mine, et d'autres non, le but étant de déminer toute la grille sans tomber sur une mine. Celui-ci éxecutable par la console de commande avec l'environnement javac, en makefile ou avec le logiciel éclipse.
== Télécharger le contenu archivé du projet ici : Démineur.zip ==

► WEB :

• BATAILLE NAVALE :
Projet du jeu de la bataille navale réalisé en ISN l'année de terminale en HTML5, CSS3 et JavaScript. Le jeu a nécessité la programmation d'une intelligence artificielle jouant en même temps que le joueur. Il contient également une musique réalisée sur le site audiotool et des images animés (sprites) réalisés sur le site Piskel.
== Vous pouvez visualiser le résultat du projet en cliquant ici ou télécharger le contenu archivé ici : Find-Sunk.zip ==
• SONDAGE.COM :
Projet d'un site de sondage programmé en php, HTML5, CSS3, mySQL et JavaScript avec codeigniter, inspiré du fonctionnement du site doodle.
== Télécharger le contenu archivé du projet ici : Sondage.com.zip ==
